Привет, у
меня была та же проблема, и я решил ее с помощью пользовательского связующего:

заклепок . связующие вещества . xlinkhref  =  function ( el , value ) {
     el . setAttributeNS ( ' http://www.w3.org/1999/xlink ' , ' href ' , ' # '  + value) 
}
HTML:

< svg  class = " icon view-icon " > 
    < use  rv-xlinkhref = " item.icon " > </ use > 
</ svg >
