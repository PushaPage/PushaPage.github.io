gulp build    сборка проэкта

gulp dev    запускает сайт на серваке

gulp babel  транскомпилмрует js в ECMAScript 2015+ и минифицирует

gulp minify  минифицирует js

gulp mincss  минифицирует css

gulp autoprefix доставляет префиксы 

gulp sprite  собирает png иконки в спрайт

gulp svg-sprite собирает svg иконки в svg-спрайт
